## Contributing
1. Clone this repo
2. Change branch `git checkout -b <YOUR NEW BRANCH NAME>`
3. Run `yarn build`
4. Run `yarn dev`
5. Make your changes
6. `git commit`, `git push` and open a [new pull request](https://github.com/Team-Surtes/Nuxt/pulls).
